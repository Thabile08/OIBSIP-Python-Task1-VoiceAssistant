# OIBSIP-Python-Task1-VoiceAssistant

A beginner-friendly Python voice assistant that listens to spoken commands and responds using speech.
It can greet the user, tell the current date and time, and perform web searches based on voice input.

## Features
Voice input with speech_recognition
Text-to-speech responses with pyttsx3
Greeting command support
Current date and time reporting
Voice-triggered web search
Error handling for unclear speech input

## Tech Stack
Python
speech_recognition
pyttsx3
datetime
webbrowser
